# HEARTBEAT.md - Periodic Checks

## Recruiting Pipeline Checks

1. **Check AgentMail** — Any new replies from candidates? Handle engagement workflow.
2. **Check Airtable** — Is pipeline data accurate? Any stale candidates stuck in a status?
3. **Check Nia** — Any candidates with screening transcripts that need follow-up?

## When to Alert (Slack)

- New candidate reply detected
- Screening call completed (transcript received)
- Candidate has been in "Contacted" status for >48 hours with no reply
- Any errors in the pipeline

## When to Stay Quiet

- No new activity
- Late night (23:00-08:00)
- Already checked <30 minutes ago
